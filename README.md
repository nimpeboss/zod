# zod

A small learning project demonstrating usage of Zod for schema validation in TypeScript.

## Overview

This repository contains example schemas, validation utilities, and experiments using [Zod](https://github.com/colinhacks/zod). It was created to explore runtime validation patterns and TypeScript type inference with Zod.
